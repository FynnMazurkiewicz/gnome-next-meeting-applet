============
Installation
============


Stable release
--------------

To install Gnome Next Meeting applet, run this command in your terminal:

.. code-block:: console

    $ pip install gnome_next_meeting_applet

This is the preferred method to install Gnome Next Meeting applet, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Gnome Next Meeting applet can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/chmouel/gnome_next_meeting_applet

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/chmouel/gnome_next_meeting_applet/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/chmouel/gnome_next_meeting_applet
.. _tarball: https://github.com/chmouel/gnome_next_meeting_applet/tarball/master
